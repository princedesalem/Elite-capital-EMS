# Frontend (React + Vite)

Minimal Vite + React scaffold.

Setup:

```bash
cd frontend
npm install
npm run dev
```

Power BI embed (Dashboard):

- Configure `VITE_POWERBI_EMBED_URL` in `frontend/.env`.
- Valeur actuelle:
	- `VITE_POWERBI_EMBED_URL=https://app.powerbi.com/view?r=REPLACE_WITH_YOUR_REPORT_ID`
- Remplace uniquement `REPLACE_WITH_YOUR_REPORT_ID` par ton lien Publish to web.
